*********************************Scheduling********************************************

This application connects to the organization's database and allow's the user to
log-in, view, create, update and delete both customer and appointment data, view
schedules by month and week, and choose from a variety of reports to view.

Author: Nick Fuller
Contact Email: nfull14@wgu.edu
Version: 1.1
Release Date: 6-30-2021
IDE: IntelliJ IDEA Community 2021.1.2
JDK: jdk-11.0.10
JavaFX: javafx-sdk-16
MySQL: mysql-connector-java-8.0.25

How to run: add the JavaFX library and the MySQL connector to the build path and
library for the project after import. The application class to run as main is
src/Scheduling.Scheduling.java.

Additional report: the additional report I chose to run in A3f is a report of all
appointments shown by First-Level Division.
